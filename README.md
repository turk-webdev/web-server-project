Project Structure
```
|-src  
  |-auth (contains auth check code)
  |-bin (contains main server code)
    |-obj (helper objects for bin code)
  |-conf (conf files to be read)
  |-parsers (code to read & parse conf files)
  |-public_html (front-end test stuff)
```
